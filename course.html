Goals

  • Understand the basics of Playwright, including its main features and capabilities
  • Set up a Playwright environment and write your first test
  • Learn how to locate elements on a page using Playwright's built-in selectors.
  • Use Playwright to interact with web pages, including clicking elements and filling out forms.
  • Write tests that can handle asynchronous events, such as network requests or JavaScript execution.
  • Debug and troubleshoot your tests when things go wrong.
  • Write tests that can run in multiple browsers, including Chrome, Firefox, and Safari.
  • Use Playwright to take screenshots and generate reports.
  • Understand the importance of test organization and learn how to structure your tests for maximum effectiveness.
  • Explore more advanced features of Playwright, including working with iframes, handling authentication, and integrating with continuous integration systems.

Description

Microsoft Playwright is one of the newest and is becoming one of the most popular E2E test automation frameworks in the industry.

Playwright is beginner-friendly, is supported with all the major browsers and can be used with multiple languages such as Typescript, Javascript, Python, Java and .NET. Best of all, it is free to use for anyone (startups to enterprise).

This course will take you from the very basics where I will teach you about what Playwright is and how to setup your first test with Playwright, to all the way to teaching how to build a fully functional framework. Along the way, I will teach you all the best practices that will put on the path of becoming one of the Top Automation Engineer.

All of the topics are taught with real examples. The course will include exercises & quizzes that will challenge you & allow you to practice what you have learned.
